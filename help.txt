FIREBASE

Para subir el proyecto al cloud
    firebase deploy --only functions

.ENV

Para coger el json con las claves: 
    firebase functions:config:get

Para guardar el json con las claves:
    cd functions
    npm run env
    cd ..
    firebase deploy --only functions
